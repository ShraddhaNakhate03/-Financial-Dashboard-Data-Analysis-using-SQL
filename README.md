# 💳 Financial Dashboard Data Analysis using SQL

## 📌 Project Overview
This project focuses on analyzing financial and credit card data using SQL to generate meaningful insights for business decision-making.  

It simulates real-world financial analytics where data is used to understand **customer behavior, transaction trends, and revenue generation**.

---

## 🎯 Objectives
- Analyze credit card transaction data  
- Understand customer demographics and behavior  
- Evaluate financial performance metrics  
- Track transaction trends over time  
- Generate insights for business optimization  

---

## 📊 Dataset Description
The project uses two main datasets:

### 1. Credit Card Details (`cc_detail`)
- Card Category  
- Annual Fees  
- Credit Limit  
- Total Transactions (Amount & Count)  
- Interest Earned  
- Utilization Ratio  
- Transaction Type  

### 2. Customer Details (`cust_detail`)
- Customer Age  
- Gender  
- Income  
- Education Level  
- Marital Status  
- State & Zipcode  
- Customer Job  
- Satisfaction Score  

---

## 🔍 Business Problems Solved

### 💳 Transaction Analysis
- Analyze total transaction amount and count  
- Track weekly and quarterly trends  

### 👥 Customer Analysis
- Segment customers based on demographics  
- Analyze income and spending behavior  

### 💰 Revenue & Interest Analysis
- Evaluate interest earned across customers  
- Analyze revenue contribution by card category  

### 📊 Utilization Insights
- Understand credit utilization patterns  
- Identify high-risk or high-value customers  

### 📈 Trend Analysis
- Analyze performance over weeks, quarters, and years

## ⚙️ How to Run

1. Create database:

 CREATE DATABASE ccdb;

2. Create tables using SQL script  

3. Import CSV data using:

COPY table_name
FROM 'file_path'
DELIMITER ','
CSV HEADER;

4. Run queries to analyze data
